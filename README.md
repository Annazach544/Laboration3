# Laboration 3 - SASS Webbplats  

## Om webbplatsen
Denna webbplats är skapad med **SASS/SCSS** och publiceras via Netlify.

Webbplatsen består av två sidor:

- **Startsida:** Visar en hero-sektion med text, galleribilder och en huvudmeny.  
- **Undersida "SASS":** Innehåller en utvärdering av arbetet med SASS, inklusive för- och nackdelar.

## Teknisk information
- **SASS/SCSS:** Variabler, mixins, nesting, partials, @use och färgfunktioner används.  
- **Dark mode & light mode:** Automatisk anpassning till användarens färgschema via `prefers-color-scheme`.  
- **Responsiv design:** Layout och meny anpassar sig efter skärmstorlek.  
- **Automatiserad arbetsprocess:** SASS kompileras till CSS med `sass --watch`.  
- **Bildhantering:** Bilder finns i `public`-mappen för enkel åtkomst.
